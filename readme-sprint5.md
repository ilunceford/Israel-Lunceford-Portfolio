# Israel Lunceford Portfolio Website

This project is a personal portfolio website that demonstrates my previous CSE 310 projects. The site includes a main portfolio page, project cards with GitHub repository links and YouTube demo links, a contact page, and JavaScript form validation for the contact form.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Download or clone the project folder.
2. Open `index.html` in a web browser.
3. Optional: run a local server from the project folder with `python -m http.server 8000`, then open `http://localhost:8000`.

Instructions for using the software:

1. Use the navigation bar to move between the project section, skills section, and contact page.
2. Click `View Repository` on any project card to open the GitHub repository.
3. Click `Watch Demo` on any project card to open the YouTube demo video.
4. Open the contact page to view my contact information or fill out the contact form.

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Visual Studio Code or another code editor
* HTML5
* CSS3
* JavaScript
* Modern web browser such as Chrome, Edge, or Firefox
* Python 3, optional for running a local test server
